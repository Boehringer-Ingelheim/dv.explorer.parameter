# Subsets a subject level dataset for the waterfall heatmap

Levels in subject column are ordered with respect to value column higher
to lower

## Usage

``` r
wfphm_wf_subset_data_cont(val_col, color_col, group_ds, subj_col)
```

## Arguments

- val_col:

  the column from the group_dataset to be used

- color_col:

  the column to be used for coloring/grouping

- group_ds:

  the subject level dataset

- subj_col:

  the subject id column
