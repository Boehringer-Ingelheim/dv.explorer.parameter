# Prepares the data for the continuous heatmap

Prepares the data for the continuous heatmap

## Usage

``` r
wfphm_hmcont_subset(data, selection, subjid_var, sorted_x)
```

## Arguments

- data:

  the dataframe to be subset, commonly a subject level dataset

- selection:

  the categorical columns to be selected

- subjid_var:

  the column that corresponde to the subject id

- sorted_x:

  the ordered subject ids
