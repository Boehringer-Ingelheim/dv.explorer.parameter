# Color bars and changes labels to outliers

`label` is replaced by `outlier`

## Usage

``` r
wfphm_wf_apply_outliers(df, ll, ul)
```

## Arguments

- df:

  the dataframe where we will apply the outliers

- ll, ul:

  the lower/upper value limit to consider an entry an outlier

## Details

`color` is replaced by `gray`
