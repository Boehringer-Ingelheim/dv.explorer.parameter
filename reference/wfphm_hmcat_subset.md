# Prepares the data for the categorical heatmap

Prepares the data for the categorical heatmap

## Usage

``` r
wfphm_hmcat_subset(data, selection, palette, subjid_var, sorted_x)
```

## Arguments

- data:

  the dataframe to be subset, commonly a subject level dataset

- selection:

  the categorical columns to be selected

- palette:

  a palette to be applied

- subjid_var:

  the column that corresponde to the subject id

- sorted_x:

  the ordered subject ids
