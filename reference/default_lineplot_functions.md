# Default lineplot summary functions

Default lineplot summary functions

## Usage

``` r
lp_mean_summary_fns

lp_median_summary_fns
```

## Format

An object of class `list` of length 3.

An object of class `list` of length 3.

## Functions

- `lp_mean_summary_fns`: Default mean functions

- `lp_median_summary_fns`: Default median functions
