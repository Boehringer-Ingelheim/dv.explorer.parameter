# Mock forestplot app

Mock forestplot app

## Usage

``` r
mock_app_forest(
  dry_run = FALSE,
  update_query_string = TRUE,
  srv_defaults = list(),
  ui_defaults = list()
)
```

## Arguments

- dry_run:

  Return parameters used in the call

- update_query_string:

  automatically update query string with app state

- ui_defaults, srv_defaults:

  a list of values passed to the ui/server function
