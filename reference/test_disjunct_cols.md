# Test has repeated cols

Test has repeated cols

## Usage

``` r
test_disjunct_cols(ds1, ds2, ignore = NULL)

need_disjunct_cols(..., msg)
```

## Arguments

- ds1, ds2:

  datframes to test

- ignore:

  columns to ignore while testing

- ...:

  Argument forwarded to test_disjunct_cols

- msg:

  Validation message
